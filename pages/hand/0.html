Überblick

In diesem Abschnitt erhältst du eine kurze Einführung in den Hand-Code-Editor.


Editor öffnen

Falls du den Editor noch nicht geöffnet hast Klicke hier.


Benutzeroberfläche

Der Aufbau erinnert an Scratch – auf der rechten Seite findest du Blöcke, die du per Drag & Drop verwenden kannst.


Steuerung (Controls)

Unter "Controls" findest du die Blöcke start und update. Es darf jeweils nur einen dieser Blöcke geben.
– start wird einmalig beim Start ausgeführt.
– update wird fortlaufend wiederholt.
Außerdem gibt es einen Block zum Warten: warten _ Sekunden.


Fingersteuerung

Im Tab „Finger“ findest du Blöcke zur Steuerung der Fingerrotation. Die Finger sind von 1 (Daumen) bis 5 (kleiner Finger) nummeriert.


Logik (Logic)

Im Bereich „Logic“ gibt es verschiedene Blöcke wie if-Bedingungen, Vergleichsblöcke sowie Elemente zur Darstellung von Zahlen und Texten.


Schleifen (Loops)

Unter „Loops“ findest du:
– repeat times: Wiederholt den enthaltenen Code eine bestimmte Anzahl von Malen.
– repeat while: Führt Code aus, solange eine Bedingung erfüllt ist.
– break: Beendet die Schleife vorzeitig.


Variablen (Variables)

Mit dem Block Create Variable kannst du eine neue Variable erstellen. Du gibst ihr einen Namen, und sie kann sich einen Wert merken – z. B. einen Wahrheitswert (bool), eine Zahl (int) oder einen Text (str).

Mit dem Block Set Variable kannst du den Wert einer Variable ändern. Mit dem Block, der nur den Namen der Variable zeigt, kannst du auf ihren aktuellen Wert zugreifen.

Hinweis: Jeder Variablenname darf nur einmal vergeben werden.


Funktionen (Functions)

Mit dem Block Function kannst du eigene Funktionen erstellen. Diese enthalten Code, der jedes Mal ausgeführt wird, wenn du die Funktion aufrufst.

Das ist besonders hilfreich, wenn du denselben Code mehrfach verwenden möchtest – du musst ihn dann nur einmal schreiben.

Sobald du eine Funktion erstellt hast, erscheint ein neuer Block, mit dem du sie aufrufen kannst.


Wie es weitergeht:

Im nächsten Schritt werden die Bauteile, die du verwenden wirst, kurz vorgestellt.
